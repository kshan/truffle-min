# truffle-min

A Script for minimizing truffle json artifact.

一个精简 Truffle 构建文件（artifact）的脚本。



## Usage
Install jq

```
brew install jq
```

run:

```
./truffle-min.sh build/contracts
```

build/contracts is the artifact dir
